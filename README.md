# Ótica Show Room - Landing Page

## Descrição
Landing page moderna e responsiva para a Ótica Show Room, desenvolvida com design sofisticado e elementos interativos.

## Funcionalidades
- Hero section com CTA "Peça um orçamento"
- Seções de produtos (masculino/feminino)
- Propaganda BlueFilter com efeito parallax
- Galeria de armações
- Sobre a empresa
- Rodapé completo com contatos

## Tecnologias Utilizadas
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- Font Awesome
- Google Fonts (Inter)

## Estrutura de Arquivos
```
/
├── index.html          # Página principal
├── hero-optica.png     # Imagem hero
├── design.md          # Documentação de design
└── README.md          # Este arquivo
```

## Design Visual
- Cores: Azul marinho (#1e3a8a), Azul vibrante (#3b82f6), Branco (#ffffff)
- Tipografia: Inter (sans-serif moderna)
- Efeitos: Glass morphism, hover animations, parallax scroll

## Responsividade
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (320px - 767px)
