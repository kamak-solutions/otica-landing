# Design Visual - Ótica Show Room

## Identidade Visual

- **Nome da marca**: Ótica Show Room
- **Setor**: Ótica / Moda / Saúde visual
- **Estilo**: Moderno, sofisticado, acessível
- **Cores principais**: Azul marinho (#1e3a8a), branco (#ffffff), cinza claro (#f8fafc)
- **Cor de destaque**: Azul vibrante (#3b82f6) para CTAs
- **Tipografia**: Sans-serif moderna (Inter, system-ui)

## Paleta de Cores

- **Primária**: Azul marinho (#1e3a8a) - profissional e confiável
- **Secundária**: Branco (#ffffff) - limpo e moderno
- **Fundo**: Cinza muito claro (#f8fafc) - suave e elegante
- **Destaque**: Azul vibrante (#3b82f6) - para botões e elementos interativos
- **Texto**: Cinza escuro (#1f2937) - alto contraste

## Tipografia

- **Títulos**: Inter Bold, tamanhos grandes (48px-64px no desktop)
- **Subtítulos**: Inter Semibold, tamanhos médios (24px-32px)
- **Corpo**: Inter Regular, tamanhos legíveis (16px-18px)
- **CTAs**: Inter Medium, tamanho destacado (18px-20px)

## Efeitos Visuais

- **Parallax**: Para a seção BlueFilter
- **Animações**: Fade-in suave nas seções
- **Hover effects**: Em cards de produtos e botões
- **Background**: Gradiente sutil no hero, padrão geométrico sutil nas seções

## Layout Responsivo

- **Desktop**: Grid de 12 colunas, containers centralizados
- **Tablet**: Grid adaptativo, manter proporções
- **Mobile**: Layout vertical, fontes redimensionadas, touch-friendly

## Elementos Interativos

- **Botões CTA**: Arredondados, sombra sutil, hover animado
- **Cards de produtos**: Hover com leve elevação
- **Menu mobile**: Hamburguer menu com animação suave
- **Formulários**: Campos modernos com foco visual
